Technical Specs

View technical specs
  • Controller: Raspberry Pi-based system
  • Display: 10.1-inch touchscreen
  • Sensor input: 12V opto-isolated sensor input
  • Line signaling: Patlite NE-M1ACNB-M multi-color signal beacon with integrated buzzer, controlled through a relay-isolated interface
  • Controls: Touchscreen increment, decrement, and reset controls
  • Networking: Ethernet connectivity for LAN dashboard access. Wi-Fi is available if needed.
  • Data: Local storage and CSV export
  • Dashboard: Local LAN dashboard with line status, notifications, recent activity, and simple line setup tools
  • Enclosure: 3D-printed enclosure with mounting hardware
  • Tested sensors: Keyence PZ-G62CB and PR-G41CBL
  • Environment: Designed for typical factory floor environments including dust, vibration, and continuous operation

FAQ

Are sensors included?

Sensors are not included, but many lines already have compatible sensors installed. If not, we can recommend options or work with your existing supplier.

What sensors has it been tested with?

Tested with Keyence PZ-G62CB and PR-G41CBL sensors.

What if my line uses a different sensor?

We can review your sensor setup with you and quote any custom sensor support before purchase.

Does it require cloud access or a subscription?

No. It runs locally on your network and does not require a subscription.

How is it powered and connected?

Ethernet is the recommended default because it simplifies local network access and can also support PoE. Wi-Fi is available if needed.

What is included in the standard system price?

The $2,495 price covers the standard system and basic setup. Custom sensor support, mounting changes, reporting changes, enclosure changes, and integrations are quoted separately.

Can it be repaired or serviced?

Yes. It is built from serviceable off-the-shelf components rather than proprietary locked-down hardware, making replacement and repair straightforward.
