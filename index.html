Real-time insights built for manufacturers on a budget. No subscriptions. No bloat.

What It Is

This compact, plug-and-play QA monitoring kit uses a Raspberry Pi, industrial sensor inputs, and a customizable dashboard to deliver real-time factory line analytics without subscription fees or unnecessary complexity.

Why It’s Better

  • No subscription required. You own the hardware, forever.
  • Custom-fit. Each unit is tailored to your exact sensor and signal specs. Out of the box, the system supports Keyence PZ-G62CB and PR-G41CBL sensors via a tested M8 adapter connected to the Raspberry Pi's GPIO.
  • Instant visibility. View widget-by-widget analytics on a built-in screen.
  • REST API or Local Dashboard. Connect to your internal systems via REST or log in directly over LAN.
  • Simpler than RedZone. No bloated software, no IT overhead.

Technical Specs

  • Processor: Raspberry Pi 4 or 5 (quad-core, 1.5–2.4GHz)
  • Display: 10” touchscreen
  • Connectivity: Wi-Fi / Ethernet (PoE optional), LAN-hosted or REST API
  • Keyence Sensor Adapter: 12V logic-compatible input via M8 connector + voltage divider or optoisolator.
  • Data Access: REST endpoint (JSON format) or local dashboard over LAN depending on your needs
  • Storage: microSD (default) with SSD upgrade available

Compatibility & Integration

  • Default Sensor Support: Keyence PZ-G62CB, PR-G41CBL (NPN, 12V logic)
  • Signal Types: NPN or 12V digital logic with included adapter
  • Connector: M8 4-pin with included custom wiring adapter

Options

Available add-ons include:

  • Multi-color beacon light for line status
  • Power-over-Ethernet integration
  • Custom dashboards and remote access setup
